Type sanskrit term in an input field above.

At any time you may use: '*' in your search to indicate 'any characters', eg:

  • 'bhoj*' will search for any word starting with 'bhoj'
  • '*vat' will search for any word ending with 'vat'
  • 'man*vat' will search for any word starting with 'man' and ending with 'vat'
  • '*kara*' will search for any word containing 'kara' at any position
"Any word ending" is equivalent to 'word*'. It is active by default.
"Any word beginning" is equivalent to '*word'.

"English" input method auto-enables "Fulltext search."
"Fulltext" can be manually toggled for "HK", "IAST", "Velthuis" and "none".

"Fulltext" looks through the whole text contained in a dictionary. Using "Fulltext" to search for words shorter than 3 characters and/or with "Any word ending/beginning" checked may result in really long searches and match list.

Chose input method, display method and search options right below input field. Use "Hide/Show options" button to toggle options visibility.
